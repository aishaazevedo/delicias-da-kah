# Delícias da Kah 🧁

Cardápio digital interativo desenvolvido para o negócio **Delícias da Kah**, facilitando a escolha de produtos e o fechamento de pedidos diretamente pelo WhatsApp.

## 🚀 Funcionalidades

* **Cardápio Dinâmico:** Categorização intuitiva de produtos (Pães, Salgados, Doces, Kits).
* **Carrinho de Compras:** Adição, visualização e totalização de itens em tempo real.
* **Finalização via WhatsApp:** Integração direta que envia o pedido formatado e permite o envio de observações personalizadas.
* **Design Responsivo:** Interface otimizada para dispositivos móveis, garantindo agilidade no atendimento.

## 🛠 Tecnologias Utilizadas

* **HTML5**
* **Tailwind CSS** (para estilização rápida e responsiva)
* **JavaScript** (lógica do carrinho, filtros e integração com WhatsApp API)
